This module is a lightweight solution to render View results as a calendar table. Compatible with any entity type and any date field type. Zero dependencies other than Views core module. 

  • Recurring events
  • Multi value date fields (i.e. unlimited cardinality)
  • Datetime range fields
  • Ajax navigation by month with a custom pager

Featured in Talking Drupal Module of the week #324. Watch the review on YouTube.
